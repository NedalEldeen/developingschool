# Developing School

This repository is an archive of the articles and blog posts I wrote and published on [developingschool.com](https://developingschool.com/) many years ago.

I originally started Developing School as both a personal blog and a small community for people who were interested in technology, software, programming, and similar topics.

Now that the original website is being shut down, I've decided to preserve that work here on GitHub.
